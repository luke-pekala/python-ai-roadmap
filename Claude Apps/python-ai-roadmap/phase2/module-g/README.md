# Module G - Sales EDA (NumPy + Pandas)

**Phase 2 - Weeks 9-10**

EDA on 18 months of synthetic sales data. Answers 5 business questions.

## Quick start

    uv sync --all-extras
    uv run pytest
    uv run jupyter notebook analysis.ipynb

## Business questions answered

1. Top 10 products by revenue
2. Which region generates most revenue?
3. Monthly revenue trend
4. Category revenue breakdown
5. Day-of-week ordering patterns

## Skills: NumPy * Pandas * groupby * Data cleaning * EDA * Jupyter * pytest
